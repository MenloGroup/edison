# edison

0. [install docker](https://docs.docker.com/engine/installation/)
1. Create a `.env` file with these environmental variables:
  * `HUBOT_NAME`
  * `HUBOT_SLACK_BOTNAME`
  * `HUBOT_SLACK_TOKEN`
  * `HUBOT_SLACK_TEAM`
  * `HUBOT_HEROKU_KEEPALIVE_URL`
  * `REDIS_URL`
2. `docker-compose up --build`
